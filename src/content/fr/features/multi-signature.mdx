## Vue d'ensemble

La technologie multi-signature (multi-sig) ajoute une couche supplémentaire de sécurité à votre TBC Wallet en exigeant plusieurs approbations avant qu'une transaction puisse être exécutée. Cette fonctionnalité est essentielle pour protéger les actifs de grande valeur et est idéale pour un usage personnel et les portefeuilles partagés.

## Qu'est-ce que la Multi-Signature?

La multi-signature est une fonctionnalité de sécurité qui nécessite plusieurs clés privées pour autoriser une transaction. Au lieu d'une seule clé contrôlant votre portefeuille, les portefeuilles multi-sig nécessitent un nombre prédéterminé de signatures (par exemple, 2 sur 3, 3 sur 5) pour approuver toute transaction.

### Comment ça fonctionne

1. **Configuration**: Configurez votre portefeuille multi-sig avec plusieurs détenteurs de clés
2. **Demande de Transaction**: Lorsqu'une transaction est initiée, elle nécessite une approbation
3. **Processus d'Approbation**: Les signataires désignés doivent approuver la transaction
4. **Exécution**: Une fois le nombre requis d'approbations atteint, la transaction s'exécute

## Avantages de Sécurité

### Protection Contre Point de Défaillance Unique

- **Perte de Clé**: Si une clé est perdue, vos fonds restent sécurisés
- **Prévention du Vol**: Une seule clé compromise ne peut pas vider votre portefeuille
- **Menaces Internes**: Réduit le risque d'insiders malveillants

### Contrôle Renforcé

- **Portefeuilles Partagés**: Parfait pour les entreprises ou les familles gérant des fonds partagés
- **Gouvernance**: Permet une prise de décision démocratique pour la gestion des fonds
- **Traçabilité**: Toutes les approbations sont enregistrées pour la transparence

### Conformité Réglementaire

- **Usage Corporatif**: Répond aux exigences pour la gestion de trésorerie d'entreprise
- **Conformité**: Aide à répondre aux exigences réglementaires pour la garde de fonds
- **Responsabilité**: Chaîne d'approbation claire pour les audits financiers

## Guide de Configuration

### Prérequis

Avant de configurer la multi-signature, assurez-vous d'avoir:

- Un compte TBC Wallet
- Plusieurs appareils de confiance ou détenteurs de clés
- Une compréhension du processus d'approbation

### Configuration Étape par Étape

1. **Accédez aux Paramètres Multi-Sig**: Naviguez vers les paramètres de Sécurité dans TBC Wallet
2. **Choisissez la Configuration**: Sélectionnez votre configuration multi-sig (par exemple, 2-sur-3, 3-sur-5)
3. **Ajoutez les Détenteurs de Clés**: Ajoutez les adresses ou appareils qui détiendront les clés de signature
4. **Configurez le Seuil**: Définissez le nombre minimum d'approbations requises
5. **Testez la Transaction**: Effectuez une transaction de test pour vérifier la configuration
6. **Activez**: Activez la protection multi-signature pour votre portefeuille

### Options de Configuration

#### Configuration 2-sur-3

- Nécessite 2 signatures sur 3
- Idéal pour un usage personnel avec clés de sauvegarde
- Équilibre entre sécurité et commodité

#### Configuration 3-sur-5

- Nécessite 3 signatures sur 5
- Meilleur pour les organisations ou portefeuilles partagés
- Sécurité accrue avec plus de flexibilité

#### Configurations Personnalisées

- Configurez toute configuration M-sur-N
- Adaptez à vos besoins de sécurité spécifiques
- Structures d'approbation flexibles

## Processus de Transaction

### Initier une Transaction

1. **Créer Transaction**: Rédigez la transaction dans TBC Wallet
2. **Soumettre pour Approbation**: La transaction est mise en file d'attente pour approbation multi-sig
3. **Notification**: Tous les détenteurs de clés reçoivent une notification
4. **Approbation**: Les détenteurs de clés examinent et approuvent/rejettent
5. **Exécution**: La transaction s'exécute une fois le seuil atteint

### Interface d'Approbation

- **Détails de Transaction**: Voir les informations complètes de la transaction
- **Approuver/Rejeter**: Interface d'approbation simple
- **Commentaires**: Ajoutez des notes ou commentaires aux approbations
- **Historique**: Suivez toutes les activités d'approbation

## Questions Fréquentes

### Combien de signatures ai-je besoin?

Le nombre de signatures requises dépend de votre configuration. Les configurations courantes sont 2-sur-3 ou 3-sur-5, mais vous pouvez personnaliser selon vos besoins.

### Que se passe-t-il si je perds une clé?

Si vous perdez une clé mais avez encore suffisamment de clés pour atteindre le seuil, vous pouvez continuer à fonctionner normalement. Cependant, vous devriez remplacer la clé perdue pour maintenir la redondance.

### Puis-je changer la configuration multi-sig?

Oui, vous pouvez modifier votre configuration multi-sig, mais cela nécessite généralement l'approbation des détenteurs de clés existants pour assurer la sécurité.

### Le multi-sig est-il plus lent que les transactions régulières?

Les transactions multi-sig peuvent prendre un peu plus de temps car elles nécessitent plusieurs approbations, mais la sécurité renforcée vaut le léger retard.

### Puis-je utiliser multi-sig avec des portefeuilles matériels?

Oui, TBC Wallet prend en charge l'intégration de portefeuilles matériels pour les configurations multi-signature, offrant une sécurité encore plus grande.
