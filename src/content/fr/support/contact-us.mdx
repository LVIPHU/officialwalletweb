## Sujets de Support

### Ce avec quoi Nous Pouvons Aider

Notre équipe de support peut vous aider avec :

- **Problèmes de Compte** : Configuration du portefeuille, récupération et gestion
- **Support Technique** : Problèmes de transaction, problèmes d'application
- **Préoccupations de Sécurité** : Questions de sécurité et meilleures pratiques
- **Questions sur les Fonctionnalités** : Comment utiliser les fonctionnalités du portefeuille
- **Rapports de Bugs** : Signalez les problèmes ou bugs que vous rencontrez
- **Demandes de Fonctionnalités** : Suggérez de nouvelles fonctionnalités ou améliorations

### Avant de Contacter le Support

Pour nous aider à mieux vous assister, veuillez avoir prêt :

- **Version du Portefeuille** : Votre version actuelle de TBC Wallet
- **Informations sur l'Appareil** : Type d'appareil et système d'exploitation
- **Description du Problème** : Description détaillée de votre problème
- **Captures d'Écran** : Si applicable, incluez des captures d'écran
- **Détails de Transaction** : Hash de transaction si lié à une transaction

## Avis de Sécurité

### Informations de Sécurité Importantes

- **Ne Partagez Jamais** : Ne partagez jamais votre phrase de récupération, clés privées ou mots de passe
- **Canaux Officiels Uniquement** : Utilisez uniquement les canaux officiels de TBC Wallet pour le support
- **Vérifiez les Sources** : Vérifiez toujours que vous contactez l'équipe officielle de TBC Wallet
- **Signalez les Arnaques** : Signalez immédiatement toute activité suspecte

### Ce que Nous Ne Demanderons Jamais

Le support TBC Wallet ne demandera **jamais** :

- Votre phrase de récupération ou mots de récupération
- Vos clés privées
- Votre mot de passe de portefeuille
- Paiement pour les services de support

Si quelqu'un prétendant être de TBC Wallet demande ces informations, c'est une arnaque. Signalez-le immédiatement.

## Questions Fréquemment Posées

### À quelle vitesse recevrai-je une réponse ?

Nous visons à répondre à toutes les demandes dans les 24 heures. Pour les problèmes de sécurité urgents, nous priorisons et répondons aussi rapidement que possible.

### Puis-je obtenir un support téléphonique ?

Actuellement, nous fournissons un support par email et nos canaux communautaires. Le support téléphonique peut être disponible pour les clients entreprise.

### Le support est-il disponible en plusieurs langues ?

Oui, notre équipe de support peut vous aider en plusieurs langues. Veuillez spécifier votre langue préférée lors du contact.

### Comment signaler un bug ?

Pour signaler un bug :

1. Envoyez-nous un email à contact@tbchatofficial.com
2. Incluez une description détaillée du problème
3. Fournissez les étapes pour reproduire le bug
4. Incluez des captures d'écran si possible
5. Spécifiez votre appareil et version de l'application
