## Resumen

La tecnología de multi-firma (multi-sig) agrega una capa adicional de seguridad a tu TB Wallet al requerir múltiples aprobaciones antes de que cualquier transacción pueda ejecutarse. Esta característica es esencial para proteger activos de alto valor y es ideal tanto para uso personal como para billeteras compartidas.

## ¿Qué es Multi-Firma?

Multi-firma es una característica de seguridad que requiere múltiples claves privadas para autorizar una transacción. En lugar de una sola clave controlando tu billetera, las billeteras multi-sig requieren un número predeterminado de firmas (por ejemplo, 2 de 3, 3 de 5) para aprobar cualquier transacción.

### Cómo Funciona

1. **Configuración**: Configura tu billetera multi-sig con múltiples titulares de claves
2. **Solicitud de Transacción**: Cuando se inicia una transacción, requiere aprobación
3. **Proceso de Aprobación**: Los firmantes designados deben aprobar la transacción
4. **Ejecución**: Una vez que se alcanza el número requerido de aprobaciones, la transacción se ejecuta

## Beneficios de Seguridad

### Protección Contra Punto Único de Falla

- **Pérdida de Clave**: Si se pierde una clave, tus fondos permanecen seguros
- **Prevención de Robo**: Una sola clave comprometida no puede vaciar tu billetera
- **Amenazas Internas**: Reduce el riesgo de insiders maliciosos

### Control Mejorado

- **Billeteras Compartidas**: Perfecto para empresas o familias que gestionan fondos compartidos
- **Gobernanza**: Permite la toma de decisiones democrática para la gestión de fondos
- **Rastro de Auditoría**: Todas las aprobaciones se registran para transparencia

### Cumplimiento Normativo

- **Uso Corporativo**: Cumple con los requisitos para la gestión del tesoro corporativo
- **Cumplimiento**: Ayuda a cumplir con los requisitos regulatorios para la custodia de fondos
- **Responsabilidad**: Cadena de aprobación clara para auditorías financieras

## Guía de Configuración

### Requisitos Previos

Antes de configurar multi-firma, asegúrate de tener:

- Una cuenta de TB Wallet
- Múltiples dispositivos de confianza o titulares de claves
- Comprensión del proceso de aprobación

### Configuración Paso a Paso

1. **Accede a Configuración Multi-Sig**: Navega a la configuración de Seguridad en TB Wallet
2. **Elige Configuración**: Selecciona tu configuración multi-sig (por ejemplo, 2-de-3, 3-de-5)
3. **Agrega Titulares de Claves**: Agrega las direcciones o dispositivos que mantendrán las claves de firma
4. **Configura Umbral**: Establece el número mínimo de aprobaciones requeridas
5. **Prueba Transacción**: Realiza una transacción de prueba para verificar la configuración
6. **Activa**: Habilita la protección multi-firma para tu billetera

### Opciones de Configuración

#### Configuración 2-de-3

- Requiere 2 de 3 firmas
- Ideal para uso personal con claves de respaldo
- Equilibrio entre seguridad y conveniencia

#### Configuración 3-de-5

- Requiere 3 de 5 firmas
- Mejor para organizaciones o billeteras compartidas
- Mayor seguridad con más flexibilidad

#### Configuraciones Personalizadas

- Configura cualquier configuración M-de-N
- Adapta a tus necesidades de seguridad específicas
- Estructuras de aprobación flexibles

## Proceso de Transacción

### Iniciar una Transacción

1. **Crear Transacción**: Redacta la transacción en TB Wallet
2. **Enviar para Aprobación**: La transacción se pone en cola para aprobación multi-sig
3. **Notificación**: Todos los titulares de claves reciben notificación
4. **Aprobación**: Los titulares de claves revisan y aprueban/rechazan
5. **Ejecución**: La transacción se ejecuta una vez que se alcanza el umbral

### Interfaz de Aprobación

- **Detalles de Transacción**: Ver información completa de la transacción
- **Aprobar/Rechazar**: Interfaz de aprobación simple
- **Comentarios**: Agregar notas o comentarios a las aprobaciones
- **Historial**: Rastrear todas las actividades de aprobación

## Preguntas Frecuentes

### ¿Cuántas firmas necesito?

El número de firmas requeridas depende de tu configuración. Las configuraciones comunes son 2-de-3 o 3-de-5, pero puedes personalizar según tus necesidades.

### ¿Qué pasa si pierdo una clave?

Si pierdes una clave pero aún tienes suficientes claves para cumplir con el umbral, puedes continuar operando normalmente. Sin embargo, debes reemplazar la clave perdida para mantener la redundancia.

### ¿Puedo cambiar la configuración multi-sig?

Sí, puedes modificar tu configuración multi-sig, pero esto típicamente requiere aprobación de los titulares de claves existentes para asegurar la seguridad.

### ¿Es multi-sig más lento que las transacciones regulares?

Las transacciones multi-sig pueden tomar un poco más de tiempo ya que requieren múltiples aprobaciones, pero la seguridad mejorada vale la mínima demora.

### ¿Puedo usar multi-sig con billeteras de hardware?

Sí, TB Wallet soporta integración de billeteras de hardware para configuraciones multi-firma, proporcionando aún mayor seguridad.
