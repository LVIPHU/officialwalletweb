## 概述

TBC Wallet 的礦工功能使您能夠參與權益證明 (PoS) 共識機制。質押您的代幣，為驗證器投票，管理節點，並通過安全且用戶友好的介面賺取獎勵。

## 1. 質押和驗證器模組

### 我的驗證器

<Image src='/assets/images/miners/my-validator.webp' alt='my-validator' width='1572' height='4000' />

在 **我的驗證器** 部分查看您當前正在質押的所有驗證器。此儀表板提供您的質押頭寸、總質押金額和當前獎勵的概覽。

### 驗證器詳情

<Image src='/assets/images/miners/validator-detail.webp' alt='validator-detail' width='1572' height='3396' />

每個驗證器顯示全面的資訊：

- **節點名稱/ID**：驗證器節點的唯一識別碼
- **佣金率**：驗證器收取的獎勵百分比
- **總質押**：所有委託者質押的代幣總金額
- **性能指標**：正常運行時間百分比和區塊簽名成功率
- **獎勵歷史**：歷史獎勵分配數據

### 質押選項

#### 普通節點到 Com.Node 質押

<Image
  src='/assets/images/miners/common-node-to-com-node-staking.webp'
  alt='common-node-to-com-node-staking'
  width='1572'
  height='4700'
/>

將您的代幣質押到普通節點 (Com.Node) 或常規驗證器節點：

1. 導航到驗證器列表
2. 選擇您想要質押的驗證器
3. 查看驗證器詳情和佣金率
4. 輸入您想要質押的金額
5. 確認交易

#### 輕節點詳情 / Com.Node 詳情

<Image src='/assets/images/miners/light-node-detail.webp' alt='light-node-detail' width='1572' height='3328' />
<Image src='/assets/images/miners/com-node-detail.webp' alt='com-node-detail' width='1572' height='4932' />

查看特定節點類型的詳細資訊：

- **輕節點**：具有優化資源使用的輕量級驗證器節點
- **Com.Node**：作為網絡中標準驗證器的普通節點

兩種節點類型都顯示：

- 總質押金額
- 當前獎勵率
- 運行狀態（活躍/非活躍）
- 性能統計

### 取消質押

#### 輕節點取消質押 / Com.Node 取消質押

<Image src='/assets/images/miners/light-node-unstake.webp' alt='light-node-unstake' width='1572' height='3460' />
<Image src='/assets/images/miners/com-node-unstake.webp' alt='com-node-unstake' width='1572' height='3460' />

要提取您質押的代幣：

1. 轉到 **我的驗證器** 部分
2. 選擇您想要取消質押的驗證器
3. 點擊 **取消質押** 並輸入金額
4. 確認取消質押交易
5. 等待解綁期（如適用）

**注意**：取消質押可能需要解綁期，然後代幣才能用於提取。

### 創建節點

<Image src='/assets/images/miners/create-node.webp' alt='create-node' width='1572' height='3460' />

通過創建您自己的節點成為驗證器：

1. 導航到 **創建節點** 部分
2. 提供節點配置詳情
3. 質押所需的最低金額
4. 設置您的佣金率
5. 提交節點創建請求

**要求**：

- 最低質押金額
- 穩定的服務器基礎設施
- 有效的節點配置
- 安全最佳實踐

### 工作流程

**驗證器列表** → **驗證器詳情** → **質押/取消質押** → **我的驗證器**

## 3. 節點管理模組

### 輕節點詳情

輕節點是專為效率設計的優化驗證器節點：

- **配置**：節點設置和技術規格
- **質押金額**：在此節點上質押的代幣總數
- **性能指標**：正常運行時間、區塊簽名率、延遲
- **運行狀態**：活躍、非活躍或維護模式

### Com.Node 詳情

普通節點作為標準驗證器：

- **節點資訊**：詳細的節點規格
- **質押池**：所有委託者的總質押
- **獎勵分配**：當前獎勵率和歷史
- **網絡位置**：在所有驗證器中的排名

### 質押介面

#### 普通節點到 Com.Node 質押

在不同節點類型之間轉換或質押：

1. 選擇源節點（普通節點）
2. 選擇目標 Com.Node
3. 輸入質押金額
4. 查看轉換率和費用
5. 確認交易

### 篩選（選擇節點）

<Image src='/assets/images/miners/filter.webp' alt='filter' width='1572' height='3408' />

使用高級篩選器找到合適的節點：

- **性能**：按正常運行時間、成功率篩選
- **獎勵**：按獎勵率或佣金排序
- **狀態**：活躍、非活躍或新節點
- **質押金額**：最低/最高質押要求
- **佣金**：從低到高的佣金率

### 工作流程

**節點列表** → **（篩選）** → **節點詳情** → **質押/取消質押**

## 4. 獎勵和礦工歷史模組

### 礦工歷史

跟踪您的質押和驗證活動：

#### 獎勵歷史

- **每日獎勵**：按日細分的獎勵
- **每區塊獎勵**：從區塊驗證中獲得的收益
- **累計總額**：所有獲得的獎勵總和
- **提取歷史**：獎勵提取記錄

#### 區塊簽名歷史

- **已簽名區塊**：您已驗證的區塊列表
- **成功率**：成功區塊簽名的百分比
- **區塊獎勵**：每個已簽名區塊的收益
- **網絡貢獻**：您對網絡安全的影響

#### 驗證器性能

- **正常運行時間統計**：驗證器活躍的時間百分比
- **獎勵效率**：每個質押代幣的平均獎勵
- **比較**：您的性能與網絡平均值的對比

### 跟踪功能

- 實時獎勵更新
- 歷史性能圖表
- 可導出的交易歷史
- 基於當前費率的獎勵預測

## 5. 錢包和代幣模組

### 代幣詳情

查看質押代幣的詳細資訊：

- **代幣餘額**：您錢包中可用的代幣
- **質押金額**：當前鎖定在質押中的代幣
- **可用於質押**：未質押的代幣，隨時可用
- **獎勵餘額**：已累積但未提取的獎勵

### 提取

#### 提取獎勵

<Image src='/assets/images/miners/withdrawal.webp' alt='withdrawal' width='1572' height='3544' />

提取您賺取的質押獎勵：

1. 導航到 **提取** 部分
2. 查看可用的獎勵餘額
3. 輸入提取金額
4. 檢查交易費用
5. 確認提取

#### 提取質押代幣

取消質押後，提取您的代幣：

1. 確保解綁期已過（如適用）
2. 轉到 **提取** 部分
3. 選擇要提取的質押代幣
4. 輸入金額
5. 確認交易

### 代幣管理

- **餘額跟踪**：監控所有代幣餘額
- **交易歷史**：所有交易的完整記錄
- **獎勵計算**：獎勵累積的詳細細分
- **多代幣支持**：管理多種代幣類型

### 工作流程

**代幣詳情** → **提取** → **交易確認**

## 開始使用

### 先決條件

在開始 PoS 挖礦之前：

- 具有足夠代幣餘額的 TBC Wallet 帳戶
- 了解 PoS 質押機制
- 穩定的互聯網連接
- 區塊鏈驗證的基礎知識

### 分步指南

1. **打開 TBC Wallet**：啟動應用程式
2. **導航到礦工**：轉到功能部分並選擇"礦工"
3. **探索驗證器**：瀏覽可用的驗證器和節點
4. **查看性能**：檢查驗證器指標和佣金率
5. **質押代幣**：選擇驗證器並質押您的代幣
6. **監控獎勵**：實時跟踪您的質押獎勵
7. **參與治理**：投票提案和驗證器選舉
8. **管理您的頭寸**：調整質押、取消質押或創建您自己的節點

### 最佳實踐

- **分散質押**：不要將所有代幣質押給單個驗證器
- **監控性能**：定期檢查驗證器的正常運行時間和獎勵
- **保持資訊更新**：了解網絡治理提案
- **安全**：永遠不要分享您的私鑰或恢復短語
- **研究**：在質押大額之前研究驗證器
- **解綁期**：在取消質押之前了解解綁期

## 常見問題

### 使用 TBC Wallet 進行 PoS 質押安全嗎？

是的，通過 TBC Wallet 進行質押是安全的。所有交易都經過加密，您的私鑰保留在您的設備上。但是，質押具有固有風險，包括驗證器削減和網絡風險。

### 獎勵多久分發一次？

獎勵分發取決於網絡和驗證器。大多數 PoS 網絡在驗證區塊時持續分發獎勵，可隨時提取。

### 我可以與多個驗證器質押嗎？

是的，您可以同時與多個驗證器質押，以分散風險並可能增加獎勵。

### 最低質押金額是多少？

最低質押金額因網絡和驗證器而異。在質押之前檢查各個驗證器的要求。

### 如何提取我的質押獎勵？

獎勵可以隨時通過提取部分提取。質押的代幣在提取之前可能需要解綁期。

### 如果驗證器被削減會發生什麼？

如果您質押的驗證器被削減，您可能會按比例損失一部分質押代幣。在質押之前始終研究驗證器。

### 質押後我可以更換驗證器嗎？

是的，您可以從一個驗證器取消質押並與另一個驗證器質押。請注意可能適用的任何解綁期。
