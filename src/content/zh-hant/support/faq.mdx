## 一般問題

### 什麼是 TBC Wallet？

TBC Wallet 是一個安全的多鏈加密貨幣錢包，支援多個區塊鏈網路，包括 TBChain、Bitcoin、Ethereum 和 Tron。它提供了一個用戶友好的介面來管理數位資產、執行交易和存取 DeFi 服務。

### TBC Wallet 支援哪些平台？

TBC Wallet 可在以下平台使用：

- **行動端**：iOS 和 Android 應用程式
- **桌面端**：Windows 和 macOS 應用程式
- **瀏覽器**：Chrome、Firefox 和 Edge 的瀏覽器擴充功能

### TBC Wallet 是免費的嗎？

是的，TBC Wallet 可以免費下載和使用。但是，區塊鏈網路會收取處理交易的交易費用，這是所有錢包的標準做法。

### 如何開始使用 TBC Wallet？

1. 從 App Store、Google Play 或我們的網站下載 TBC Wallet
2. 建立新錢包或匯入現有錢包
3. 使用強密碼保護您的錢包並備份您的恢復短語
4. 開始管理您的數位資產

## 安全性

### TBC Wallet 有多安全？

TBC Wallet 採用多層安全措施：

- **加密儲存**：所有私鑰都經過加密並本地儲存在您的裝置上
- **多重簽名支援**：為高價值交易提供增強的安全性
- **生物識別認證**：支援指紋和面部識別
- **定期安全更新**：持續的安全改進

### 如果我遺失了裝置該怎麼辦？

如果您遺失了裝置：

1. 在新裝置上使用您的恢復短語來恢復您的錢包
2. 如果可能，立即將資金轉移到新錢包
3. 如需幫助，請聯繫支援
4. 永遠不要與任何人分享您的恢復短語

### 如何備份我的錢包？

始終在安全位置備份您的恢復短語（12 或 24 個詞）：

- 寫在紙上並安全保存
- 永遠不要以數位方式儲存或在線分享
- 考慮使用硬體錢包以獲得額外的安全性

### TBC Wallet 可以存取我的資金嗎？

不可以，TBC Wallet 無法存取您的資金。您的私鑰本地儲存在您的裝置上，永遠不會傳輸到我們的伺服器。您完全控制您的資產。

## 功能

### 支援哪些區塊鏈網路？

TBC Wallet 支援：

- TBChain（原生網路）
- Bitcoin
- Ethereum 和 EVM 相容鏈
- Tron
- 以及不斷添加的更多網路

### 什麼是多重簽名？

多重簽名（multi-sig）要求在交易執行前獲得多個批准。這增加了額外的安全層，非常適合共享錢包或高價值帳戶。

### 即時轉帳如何運作？

即時轉帳允許以最低費用進行快速跨鏈交易。它使用先進的跨鏈協議在不同區塊鏈網路之間幾乎即時地轉移資產。

### 我可以用 USDT 支付費用嗎？

是的，TBC Wallet 支援使用 USDT（Tether）支付交易費用，提供價格穩定性和可預測性，在市場波動條件下尤其有價值。

## 交易

### 交易需要多長時間？

交易時間因網路而異：

- **TBChain**：通常在幾秒鐘內完成
- **Bitcoin**：10-60 分鐘，取決於網路擁堵情況
- **Ethereum**：15 秒到幾分鐘
- **Tron**：通常在不到一分鐘內完成

### 什麼是交易費用？

交易費用是處理交易的網路費用。費用因以下因素而異：

- 網路擁堵
- 交易複雜性
- 選擇的優先級級別

### 我可以取消交易嗎？

一旦交易在區塊鏈上得到確認，就無法取消。在確認前請始終驗證交易詳情。

### 為什麼我的交易處於待處理狀態？

交易可能因以下原因處於待處理狀態：

- 網路擁堵
- 交易費用低
- 網路維護
- 餘額不足

## 故障排除

### 我的交易失敗了。我該怎麼辦？

如果交易失敗：

1. 檢查您的網路連接
2. 驗證您是否有足夠的餘額支付費用
3. 嘗試增加交易費用
4. 如果問題持續，請聯繫支援

### 我看不到我的餘額。出了什麼問題？

如果您的餘額沒有顯示：

1. 檢查您的網際網路連接
2. 刷新錢包
3. 驗證您是否在正確的網路上
4. 在區塊鏈瀏覽器上檢查交易是否成功

### 應用程式無法正常運作。我能做什麼？

嘗試以下步驟：

1. 更新到最新版本
2. 清除應用程式快取
3. 重新啟動應用程式
4. 如有必要，重新安裝（確保您有恢復短語）

### 如何更新 TBC Wallet？

更新可通過以下方式獲得：

- **行動端**：App Store 或 Google Play Store
- **桌面端**：從我們的網站下載
- **瀏覽器擴充功能**：通過瀏覽器的擴充功能商店更新

## 帳戶管理

### 如何更改我的密碼？

1. 轉到設定
2. 選擇安全
3. 選擇更改密碼
4. 輸入您當前和新密碼

是的，您可以在 TBC Wallet 中建立和管理多個錢包。每個錢包都有自己的恢復短語和私鑰。

### 如何刪除我的錢包？

要刪除錢包：

1. 轉到設定
2. 選擇錢包管理
3. 選擇要刪除的錢包
4. 確認刪除（確保您已備份恢復短語）
