## 一般问题

### 什么是 TBC Wallet？

TBC Wallet 是一个安全的多链加密货币钱包，支持多个区块链网络，包括 TBChain、Bitcoin、Ethereum 和 Tron。它提供了一个用户友好的界面来管理数字资产、执行交易和访问 DeFi 服务。

### TBC Wallet 支持哪些平台？

TBC Wallet 可在以下平台使用：

- **移动端**：iOS 和 Android 应用程序
- **桌面端**：Windows 和 macOS 应用程序
- **浏览器**：Chrome、Firefox 和 Edge 的浏览器扩展

### TBC Wallet 是免费的吗？

是的，TBC Wallet 可以免费下载和使用。但是，区块链网络会收取处理交易的交易费用，这是所有钱包的标准做法。

### 如何开始使用 TBC Wallet？

1. 从 App Store、Google Play 或我们的网站下载 TBC Wallet
2. 创建新钱包或导入现有钱包
3. 使用强密码保护您的钱包并备份您的恢复短语
4. 开始管理您的数字资产

## 安全性

### TBC Wallet 有多安全？

TBC Wallet 采用多层安全措施：

- **加密存储**：所有私钥都经过加密并本地存储在您的设备上
- **多重签名支持**：为高价值交易提供增强的安全性
- **生物识别认证**：支持指纹和面部识别
- **定期安全更新**：持续的安全改进

### 如果我丢失了设备该怎么办？

如果您丢失了设备：

1. 在新设备上使用您的恢复短语来恢复您的钱包
2. 如果可能，立即将资金转移到新钱包
3. 如需帮助，请联系支持
4. 永远不要与任何人分享您的恢复短语

### 如何备份我的钱包？

始终在安全位置备份您的恢复短语（12 或 24 个词）：

- 写在纸上并安全保存
- 永远不要以数字方式存储或在线分享
- 考虑使用硬件钱包以获得额外的安全性

### TBC Wallet 可以访问我的资金吗？

不可以，TBC Wallet 无法访问您的资金。您的私钥本地存储在您的设备上，永远不会传输到我们的服务器。您完全控制您的资产。

## 功能

### 支持哪些区块链网络？

TBC Wallet 支持：

- TBChain（原生网络）
- Bitcoin
- Ethereum 和 EVM 兼容链
- Tron
- 以及不断添加的更多网络

### 什么是多重签名？

多重签名（multi-sig）要求在交易执行前获得多个批准。这增加了额外的安全层，非常适合共享钱包或高价值账户。

### 即时转账如何工作？

即时转账允许以最低费用进行快速跨链交易。它使用先进的跨链协议在不同区块链网络之间几乎即时地转移资产。

### 我可以用 USDT 支付费用吗？

是的，TBC Wallet 支持使用 USDT（Tether）支付交易费用，提供价格稳定性和可预测性，在市场波动条件下尤其有价值。

## 交易

### 交易需要多长时间？

交易时间因网络而异：

- **TBChain**：通常在几秒钟内完成
- **Bitcoin**：10-60 分钟，取决于网络拥堵情况
- **Ethereum**：15 秒到几分钟
- **Tron**：通常在不到一分钟内完成

### 什么是交易费用？

交易费用是处理交易的网络费用。费用因以下因素而异：

- 网络拥堵
- 交易复杂性
- 选择的优先级级别

### 我可以取消交易吗？

一旦交易在区块链上得到确认，就无法取消。在确认前请始终验证交易详情。

### 为什么我的交易处于待处理状态？

交易可能因以下原因处于待处理状态：

- 网络拥堵
- 交易费用低
- 网络维护
- 余额不足

## 故障排除

### 我的交易失败了。我该怎么办？

如果交易失败：

1. 检查您的网络连接
2. 验证您是否有足够的余额支付费用
3. 尝试增加交易费用
4. 如果问题持续，请联系支持

### 我看不到我的余额。出了什么问题？

如果您的余额没有显示：

1. 检查您的互联网连接
2. 刷新钱包
3. 验证您是否在正确的网络上
4. 在区块链浏览器上检查交易是否成功

### 应用程序无法正常工作。我能做什么？

尝试以下步骤：

1. 更新到最新版本
2. 清除应用程序缓存
3. 重新启动应用程序
4. 如有必要，重新安装（确保您有恢复短语）

### 如何更新 TBC Wallet？

更新可通过以下方式获得：

- **移动端**：App Store 或 Google Play Store
- **桌面端**：从我们的网站下载
- **浏览器扩展**：通过浏览器的扩展商店更新

## 账户管理

### 如何更改我的密码？

1. 转到设置
2. 选择安全
3. 选择更改密码
4. 输入您当前和新密码
5. 确认更改

### 我可以拥有多个钱包吗？

是的，您可以在 TBC Wallet 中创建和管理多个钱包。每个钱包都有自己的恢复短语和私钥。

### 如何删除我的钱包？

要删除钱包：

1. 转到设置
2. 选择钱包管理
3. 选择要删除的钱包
4. 确认删除（确保您已备份恢复短语）
