## 概述

TBC Wallet 的矿工功能使您能够参与权益证明 (PoS) 共识机制。质押您的代币，为验证器投票，管理节点，并通过安全且用户友好的界面赚取奖励。

## 1. 质押和验证器模块

### 我的验证器

<Image src='/assets/images/miners/my-validator.webp' alt='my-validator' width='1572' height='4000' />

在 **我的验证器** 部分查看您当前正在质押的所有验证器。此仪表板提供您的质押头寸、总质押金额和当前奖励的概览。

### 验证器详情

<Image src='/assets/images/miners/validator-detail.webp' alt='validator-detail' width='1572' height='3396' />

每个验证器显示全面的信息：

- **节点名称/ID**：验证器节点的唯一标识符
- **佣金率**：验证器收取的奖励百分比
- **总质押**：所有委托者质押的代币总金额
- **性能指标**：正常运行时间百分比和区块签名成功率
- **奖励历史**：历史奖励分配数据

### 质押选项

#### 普通节点到 Com.Node 质押

<Image
  src='/assets/images/miners/common-node-to-com-node-staking.webp'
  alt='common-node-to-com-node-staking'
  width='1572'
  height='4700'
/>

将您的代币质押到普通节点 (Com.Node) 或常规验证器节点：

1. 导航到验证器列表
2. 选择您想要质押的验证器
3. 查看验证器详情和佣金率
4. 输入您想要质押的金额
5. 确认交易

#### 轻节点详情 / Com.Node 详情

<Image src='/assets/images/miners/light-node-detail.webp' alt='light-node-detail' width='1572' height='3328' />
<Image src='/assets/images/miners/com-node-detail.webp' alt='com-node-detail' width='1572' height='4932' />

查看特定节点类型的详细信息：

- **轻节点**：具有优化资源使用的轻量级验证器节点
- **Com.Node**：作为网络中标准验证器的普通节点

两种节点类型都显示：

- 总质押金额
- 当前奖励率
- 运行状态（活跃/非活跃）
- 性能统计

### 取消质押

#### 轻节点取消质押 / Com.Node 取消质押

<Image src='/assets/images/miners/light-node-unstake.webp' alt='light-node-unstake' width='1572' height='3460' />
<Image src='/assets/images/miners/com-node-unstake.webp' alt='com-node-unstake' width='1572' height='3460' />

要提取您质押的代币：

1. 转到 **我的验证器** 部分
2. 选择您想要取消质押的验证器
3. 点击 **取消质押** 并输入金额
4. 确认取消质押交易
5. 等待解绑期（如适用）

**注意**：取消质押可能需要解绑期，然后代币才能用于提取。

### 创建节点

<Image src='/assets/images/miners/create-node.webp' alt='create-node' width='1572' height='3460' />

通过创建您自己的节点成为验证器：

1. 导航到 **创建节点** 部分
2. 提供节点配置详情
3. 质押所需的最低金额
4. 设置您的佣金率
5. 提交节点创建请求

**要求**：

- 最低质押金额
- 稳定的服务器基础设施
- 有效的节点配置
- 安全最佳实践

### 工作流程

**验证器列表** → **验证器详情** → **质押/取消质押** → **我的验证器**

## 2. 治理和投票模块

### 投票选择

<Image src='/assets/images/miners/voting-selection.webp' alt='voting-selection' width='1572' height='3460' />

通过投票提案和验证器参与网络治理：

1. 访问 **投票选择** 界面
2. 浏览可用的提案或验证器选举
3. 查看提案详情和影响
4. 选择您的投票选项

### 为 Com.Node 投票

<Image src='/assets/images/miners/vote-for-com-node.webp' alt='vote-for-com-node' width='1572' height='3544' />

在验证器选举中为普通节点投票：

1. 导航到 **为 Com.Node 投票**
2. 选择您想要支持的节点
3. 查看节点的记录和性能
4. 确认您的投票
5. 监控投票进度

### 投票详情

<Image
  src='/assets/images/miners/voting-selection-detail.webp'
  alt='voting-selection-detail'
  width='1572'
  height='3460'

- **提案影响**：如果提案通过，预期的影响

### 投票记录

#### 投票记录详情 / Com.Node 投票记录详情

- **提案状态**：已投票提案的当前状态
- **投票结果**：过去投票的结果
- **奖励影响**：您的投票如何影响奖励

### 取消投票

#### 取消投票警告

<Image src='/assets/images/miners/cancel-vote-warning.webp' alt='cancel-vote-warning' width='1572' height='3460' />

在取消投票之前，请查看：

- **影响**：取消如何影响投票结果
- **截止日期**：您是否在取消窗口内

**投票选择** → **为 Com.Node 投票** → **投票详情** → **投票记录详情**

## 3. 节点管理模块

### 轻节点详情

轻节点是专为效率设计的优化验证器节点：

- **配置**：节点设置和技术规格
- **质押金额**：在此节点上质押的代币总数
- **性能指标**：正常运行时间、区块签名率、延迟
- **运行状态**：活跃、非活跃或维护模式

### Com.Node 详情

普通节点作为标准验证器：

- **节点信息**：详细的节点规格
- **质押池**：所有委托者的总质押
- **奖励分配**：当前奖励率和历史
- **网络位置**：在所有验证器中的排名

### 质押界面

#### 普通节点到 Com.Node 质押

在不同节点类型之间转换或质押：

1. 选择源节点（普通节点）
2. 选择目标 Com.Node
3. 输入质押金额
4. 查看转换率和费用
5. 确认交易

### 筛选（选择节点）

<Image src='/assets/images/miners/filter.webp' alt='filter' width='1572' height='3408' />

使用高级筛选器找到合适的节点：

- **性能**：按正常运行时间、成功率筛选
- **奖励**：按奖励率或佣金排序
- **状态**：活跃、非活跃或新节点
- **质押金额**：最低/最高质押要求
- **佣金**：从低到高的佣金率

### 工作流程

**节点列表** → **（筛选）** → **节点详情** → **质押/取消质押**

## 4. 奖励和矿工历史模块

### 矿工历史

跟踪您的质押和验证活动：

#### 奖励历史

- **每日奖励**：按日细分的奖励
- **每区块奖励**：从区块验证中获得的收益
- **累计总额**：所有获得的奖励总和
- **提取历史**：奖励提取记录

#### 区块签名历史

- **已签名区块**：您已验证的区块列表
- **成功率**：成功区块签名的百分比
- **区块奖励**：每个已签名区块的收益
- **网络贡献**：您对网络安全的影响

#### 验证器性能

- **正常运行时间统计**：验证器活跃的时间百分比
- **奖励效率**：每个质押代币的平均奖励
- **比较**：您的性能与网络平均值的对比

### 跟踪功能

- 实时奖励更新
- 历史性能图表
- 可导出的交易历史
- 基于当前费率的奖励预测

## 5. 钱包和代币模块

### 代币详情

查看质押代币的详细信息：

- **代币余额**：您钱包中可用的代币
- **质押金额**：当前锁定在质押中的代币
- **可用于质押**：未质押的代币，随时可用
- **奖励余额**：已累积但未提取的奖励

### 提取

#### 提取奖励

<Image src='/assets/images/miners/withdrawal.webp' alt='withdrawal' width='1572' height='3544' />

提取您赚取的质押奖励：

1. 导航到 **提取** 部分
2. 查看可用的奖励余额
3. 输入提取金额
4. 检查交易费用
5. 确认提取

#### 提取质押代币

取消质押后，提取您的代币：

1. 确保解绑期已过（如适用）
2. 转到 **提取** 部分
3. 选择要提取的质押代币
4. 输入金额
5. 确认交易

### 代币管理

- **余额跟踪**：监控所有代币余额
- **交易历史**：所有交易的完整记录
- **奖励计算**：奖励累积的详细细分
- **多代币支持**：管理多种代币类型

### 工作流程

**代币详情** → **提取** → **交易确认**

## 开始使用

### 先决条件

在开始 PoS 挖矿之前：

- 具有足够代币余额的 TBC Wallet 账户
- 了解 PoS 质押机制
- 稳定的互联网连接
- 区块链验证的基础知识

### 分步指南

1. **打开 TBC Wallet**：启动应用程序
2. **导航到矿工**：转到功能部分并选择"矿工"
3. **探索验证器**：浏览可用的验证器和节点
4. **查看性能**：检查验证器指标和佣金率
5. **质押代币**：选择验证器并质押您的代币
6. **监控奖励**：实时跟踪您的质押奖励
7. **参与治理**：投票提案和验证器选举
8. **管理您的头寸**：调整质押、取消质押或创建您自己的节点

### 最佳实践

- **分散质押**：不要将所有代币质押给单个验证器
- **监控性能**：定期检查验证器的正常运行时间和奖励
- **保持信息更新**：了解网络治理提案
- **安全**：永远不要分享您的私钥或恢复短语
- **研究**：在质押大额之前研究验证器
- **解绑期**：在取消质押之前了解解绑期

## 常见问题

### 使用 TBC Wallet 进行 PoS 质押安全吗？

是的，通过 TBC Wallet 进行质押是安全的。所有交易都经过加密，您的私钥保留在您的设备上。但是，质押具有固有风险，包括验证器削减和网络风险。

### 奖励多久分发一次？

奖励分发取决于网络和验证器。大多数 PoS 网络在验证区块时持续分发奖励，可随时提取。

### 我可以与多个验证器质押吗？

是的，您可以同时与多个验证器质押，以分散风险并可能增加奖励。

### 最低质押金额是多少？

最低质押金额因网络和验证器而异。在质押之前检查各个验证器的要求。

### 如何提取我的质押奖励？

奖励可以随时通过提取部分提取。质押的代币在提取之前可能需要解绑期。

### 如果验证器被削减会发生什么？

如果您质押的验证器被削减，您可能会按比例损失一部分质押代币。在质押之前始终研究验证器。

### 质押后我可以更换验证器吗？

是的，您可以从一个验证器取消质押并与另一个验证器质押。请注意可能适用的任何解绑期。
